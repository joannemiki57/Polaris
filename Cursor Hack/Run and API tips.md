# Run and API tips

From repo root (`Cursor Hackaton/`):

```bash
npm install
cp server/.env.example server/.env   # then edit keys
npm run dev
```

- **UI**: http://localhost:5173
- **API**: http://localhost:8787 — try http://localhost:8787/api/health (root `/` is a small landing page).
